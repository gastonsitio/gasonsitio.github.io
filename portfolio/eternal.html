Eternal fragancias

Diseño de un sistema de identidad visual para Eternal, una marca de perfumes y fragancias hechas en Argentina. El trabajo consiste de un sistema de identidad desde el cual parten todas las gráficas de comunicación.

Uno de los objetivos para este diseño era la búsqueda de la mayor eficiencia posible para los insumos de impresión, así como de producción de imágenes para el ámbito digital.

El diseño de packaging consta de una caja genérica para todos los productos, con el pegado de una etiqueta que dobla en dos caras para identificar la fragancia. Siendo el color un elemento clave para la identificación visual del producto. También se le suma otra etiqueta genérica con el nombre de la fragancia que va sobre el frasco.

La visualización del producto es un render realizado en el software Blender.

Esta lógica del sistema se repite con el diseño de la tienda web y el perfil en instagram. Donde se aprecia la aplicación de los elementos del sistema y sus colores, la tipografía y la estructura compositiva.

De esta manera buscamos la facilidad de replicar el sistema para crear nuevas imágenes, siendo relativamente pocas las decisiones de diseño a tomar para cada operación.
